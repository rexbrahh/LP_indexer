## Quickstart (local)

```bash
# 1) Enter the dev shell (recommended)
nix develop

# 2) Start local dependencies (NATS + ClickHouse)
make up

# 3) Create JetStream stream/consumer (requires `nats` CLI)
make ops.jetstream.init

# 4) Run a pipeline component (example: geyser ingestor)
make run.ingestor.geyser
```

If you want a self-contained validation harness, see:

- [Ops: Local dev](./ops/local-dev)
- `make sink-e2e` (ClickHouse + Parquet sink end-to-end harness)
